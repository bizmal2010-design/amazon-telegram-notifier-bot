# Amazon Telegram Notification Bot (Python + Telegram API)

An automated notification system that integrates with price trackers to send instant updates and alerts directly to a Telegram channel or chat whenever a price drop or stock change is detected on Amazon.

## Features
- **Instant Alerts:** Sends real-time messages with product links, old prices, and new prices within 3 seconds.
- **Reliable Integration:** Seamlessly connects with database logs (SQLite/Excel) to compare price shifts.
- **24/7 Monitoring Automation:** Operates continuously as a lightweight background service to ensure you never miss a deal.
